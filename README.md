
# Market Performance Impact Analysis

## Project Overview
Analyzed stock price trends over 5+ years to generate data-driven investment insights.

## Tools Used
- Power BI – Dashboard creation & data visualization
- Power Query – Data cleaning & transformation
- DAX (Basic) – Calculated measures & reporting

## Key Highlights
- Built interactive dashboards to track stock performance trends
- Identified data anomalies and documented findings with recommendations
- Simulated a compliance reporting workflow for structured decision-making

In Progress | MBA Project – Osmania University (2024–Present)
